🎲 数当てゲーム（Guess the Number）

🔍 概要

このリポジトリは、ユーザーが指定された範囲内で乱数を当てるゲーム形式のPythonスクリプトです。条件分岐、ループ、数値比較などの基本構文を学ぶのに適しています。

💡 主な機能

ランダムな整数を生成（例：1〜100）

ユーザーの入力値と比較して結果を返す

高すぎる・低すぎる・正解！などのフィードバックを表示

正解するまで繰り返し挑戦可能

🛠️ 使用技術

Python（randomモジュール・入力処理・ループ・if文）

🚀 実行方法

python guess_num.py
